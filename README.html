太閤 to Label XLSX

太閤ファイル（.TKO または .xls）からラベル一覧のExcelファイル（.xlsx）を作成するツールです。

ファイル構成

  • tko_to_label_xlsx.py: 変換処理本体
  • run_gui.bat: GUIダイアログ付きで実行するバッチファイル

.TKO、.xls および .xlsx ファイルは .gitignore により管理対象外です。

必要環境

  • Windows
  • Python 3
  • Pythonライブラリ
pip install xlrd openpyxl

GUIで実行する

run_gui.bat をダブルクリックしてください。

  1. 太閤ファイル選択ダイアログで入力 .TKO または .xls ファイルを選択します。
  2. 保存ダイアログで出力 .xlsx ファイル名を指定します。
  3. 変換が完了すると完了メッセージが表示されます。

コマンドで実行する

入力ファイルと出力ファイルを指定する場合:

python tko_to_label_xlsx.py ラベル2.TKO ラベル2.xlsx

入力ファイルだけ指定する場合:

python tko_to_label_xlsx.py ラベル2.TKO

この場合、入力ファイルと同じ名前の .xlsx が作成されます。

.xls ファイルも同じように指定できます。

python tko_to_label_xlsx.py ラベル2.xls

引数なしで実行する場合:

python tko_to_label_xlsx.py

スクリプトと同じフォルダに .TKO または .xls ファイルが1つだけある場合、そのファイルを入力として同名の .xlsx を作成します。

出力内容

出力Excelには次の列を作成します。

  • 列№
  • ラベル
  • 質問タイトル
  • タイプ
  • カテゴリ数
  • カテゴリ№
  • カテゴリ

出力時には、列幅・ヘッダー書式・罫線を自動設定します。

処理仕様

  • ｼｽﾃﾑ管理 で始まるシートを処理対象にします。
  • 複数の管理シートがある場合は、順番に処理して 列№ を通し番号で採番します。
  • 項目がない管理シートはスキップします。
  • カテゴリ数とカテゴリ名の個数を検証します。
  • 不一致がある場合は警告を表示し、エラーレポートを出力します。ただし変換処理は継続します。

注意事項

  • 出力先に既存ファイルを指定した場合は上書きされます。
  • GUI実行では保存ダイアログで上書き確認が表示されます。
  • Pythonまたは必要ライブラリが入っていない場合、変換は実行できません。
